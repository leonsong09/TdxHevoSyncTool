# 通达信 T0002 配置备份与转移工具（TdxHevoSyncTool）

用于备份/迁移 **通达信 `T0002` 目录**中的个人配置数据（PyQt6 GUI）。

## 你可以用它做什么

- 选择一个 `T0002` 目录（自动检测 + 手动选择）
- 勾选要迁移的数据项（左侧列表），右侧查看详细说明：用途、包含路径、风险提示、文件状态
- 导出：打包为 ZIP / 导出到文件夹（带 `manifest.json` 文件哈希校验）
- 导入：从 ZIP / 文件夹导入到目标 `T0002`（导入前自动备份，可回滚）
- 对比/导入（两版本）：对比两套通达信 `T0002` 差异，按策略选择单向导入或双向同步
- `user.ini`：**禁止整体复制**，仅允许通过“`user.ini 定向同步`”按 section 安全迁移设置

## 安全等级说明

- **安全（默认全选）**：通常跨版本兼容。
- **谨慎（默认不选）**：不同版本/不同券商环境可能不兼容，导入后如异常请使用回滚。
- **禁止（仅提示）**：不允许直接导入/导出（例如 `user.ini`）。

## 使用流程（推荐）

1. 关闭通达信主程序（必须）。
2. 选择源 `T0002`（或目标 `T0002`，视操作而定）。
3. 勾选需要迁移的数据项（建议先只选“安全”）。
4. 导出为 ZIP（或导出到文件夹）。
5. 在目标机器/目标版本通达信中，选择目标 `T0002`，执行导入。
6. 如导入后出现异常，可点击“回滚上次导入”恢复。

## 对比/导入（两版本）

用于“老版本 → 新版本”、或“本机 → 另一台电脑”之间的迁移与同步。

1. 打开主界面：点击 `对比/导入（两版本）`。
2. 分别选择左右两侧路径（支持选择 **安装根目录** 或 **T0002 目录**，会自动定位到 `T0002`）。
3. 点击 `开始对比`，查看各数据项在左右两侧的差异摘要与文件预览。
4. 选择模式并执行：
   - **单向导入（左→右 / 右→左）**：
     - `仅导入差异`：复制“缺失 + 冲突(不同内容)”文件
     - `仅补齐缺失`：只复制目标侧没有的文件
     - `全量覆盖`：选中数据项在源侧的全部文件覆盖到目标侧
   - **双向同步**：
     - `双向：仅补齐缺失`：两边互相补齐缺失文件，不处理冲突
     - `双向：覆盖同步`：缺失必补齐；冲突按所选规则处理（以左为准 / 以右为准 / 按修改时间较新）

> 注意：`user.ini` 永远不会参与对比导入（禁止项），请使用主界面的 `user.ini 定向同步`。

## `user.ini` 迁移规则（重要）

`user.ini` 是个性化配置核心文件，跨版本整体复制可能导致闪退。

本工具会将其标记为“禁止”，不支持直接导入/导出；请使用主界面按钮：

- `user.ini 定向同步`
  - 普通 section：只替换目标中已存在且值不同的键
  - `extern_*`：替换同名异值键，并追加目标中缺失的键
  - 目标中不存在的 section 会跳过，不会新增段落

## 开发运行

```powershell
python -m pip install -r requirements.txt
python main.py
```

## 打包（生成可双击运行的单文件 EXE）

双击运行：

- `build_gui.bat`

输出产物：

- 单文件版：`dist\TdxHevoSyncTool.exe`

## 临时目录（重要）

- 单文件 EXE 运行时会在**当前工作目录**使用 `.\temp\` 作为解包临时目录。
- “对比/导入（两版本）”会在 `.\temp\tdx_hevo_transfer\` 下创建中转目录；成功后会自动清理，失败会保留并在错误信息中给出路径便于排查。
- 请确保当前工作目录可写（必要时可手动创建 `temp` 文件夹）。
