# 通达信 T0002 配置备份与转移工具（TdxHevoSyncTool）

用于备份/迁移 **通达信 `T0002` 目录**中的个人配置数据（PyQt6 GUI）。

## 你可以用它做什么

- 选择一个 `T0002` 目录（自动检测 + 手动选择）
- 勾选要迁移的数据项（左侧列表），右侧查看详细说明：用途、包含路径、风险提示、文件状态
- 导出：打包为 ZIP / 导出到文件夹（带 `manifest.json` 文件哈希校验）
- 导入：从 ZIP / 文件夹导入到目标 `T0002`（导入前自动备份，可回滚）
- `user.ini`：**禁止整体复制**，仅允许通过“`user.ini extern 合并`”迁移 `extern_*` 段落

## 安全等级说明

- **安全（默认全选）**：通常跨版本兼容。
- **谨慎（默认不选）**：不同版本/不同券商环境可能不兼容，导入后如异常请使用回滚。
- **禁止（仅提示）**：不允许直接导入/导出（例如 `user.ini`）。

## 使用流程（推荐）

1. 关闭通达信主程序（必须）。
2. 选择源 `T0002`（或目标 `T0002`，视操作而定）。
3. 勾选需要迁移的数据项（建议先只选“安全”）。
4. 导出为 ZIP（或导出到文件夹）。
5. 在目标机器/目标版本通达信中，选择目标 `T0002`，执行导入。
6. 如导入后出现异常，可点击“回滚上次导入”恢复。

## `user.ini` 迁移规则（重要）

`user.ini` 是个性化配置核心文件，跨版本整体复制可能导致闪退。

本工具会将其标记为“禁止”，不支持直接导入/导出；请使用主界面按钮：

- `user.ini extern 合并`：仅迁移 `extern_*` 段落到新版本 `user.ini`
